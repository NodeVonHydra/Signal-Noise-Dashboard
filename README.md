# Signal Noise Dashboard

Signal Noise Dashboard is a small web app for tracking useful ideas and separating valuable information (signal) from useless information (noise).

The app helps users capture ideas, rate them by importance, track topics over time, and generate weekly intelligence briefings.

## Features

- Add information items
- Rate signal vs noise
- Track topic trends
- Generate weekly summaries
- Export and import data
- Works offline

## How to run the app

Download the project and open:

index.html

in your browser.

You can also run a local server:

python3 -m http.server

and then open:

http://localhost:8000

## Tech

This project uses simple web technologies:

- HTML
- CSS
- JavaScript
- LocalStorage

No backend is required.
